Main Components of a PC:

1. Central Processing Unit (CPU)

  • The "brain" of the computer, responsible for performing most of the computations.
  • Executes program instructions and controls all other components.
  • Major manufacturers: Intel, AMD.

2. Random Access Memory (RAM)

  • Temporary memory used to store data and instructions currently in use.
  • The data in RAM is lost when the computer is turned off.
  • The more RAM you have, the more programs you can run simultaneously without slowing down.

3. Hard Disk Drive (HDD) or Solid State Drive (SSD)

  • HDD: A storage device that uses spinning magnetic disks to store data. It is slower but cheaper and typically offers more storage capacity.
  • SSD: A faster storage device that uses flash memory. It significantly speeds up your computer (faster program and operating system loading).
  • The main circuit board that connects all components of the computer (CPU, RAM, storage, GPU, etc.).

4. Motherboard

  • The main circuit board that connects all components of the computer (CPU, RAM, storage, GPU, etc.).
  • Allows components to communicate and provides power distribution.

5. Graphics Processing Unit (GPU)

  • Handles graphics rendering and displays images on the monitor.
  • Can be integrated (built into the CPU) or discrete (separate card).
  • Discrete GPUs are essential for gaming, 3D rendering, and video editing. Major manufacturers: NVIDIA, AMD.

6. Power Supply Unit (PSU)

  • Provides electrical power to all the computer's components.
  • Converts electricity from the wall outlet into usable power for the PC.

7. Computer Case

  • Encloses and protects the internal components and aids with cooling.
  • Houses components such as the motherboard, power supply, storage drives, and more.

8. Cooling Systems

  • Fans (coolers): Used to cool the CPU, GPU, or overall system.
  • Liquid cooling (for high-performance PCs): A more complex system that reduces component temperatures during heavy loads.

9. Monitor

  • The display device that shows images, videos, and text.
  • Key specifications: resolution, screen size, refresh rate.

10. Input/Output Devices

  • Keyboard and mouse: Primary input devices.
  • Speakers or headphones: For audio output.
  • Printer, scanner: For printing or scanning documents.

11. Network Interface Card (NIC)

  • Allows the computer to connect to the internet or local network.
  • Can be integrated into the motherboard or installed as a separate card.

Software Components:

1. Operating System (OS)

  • Manages the interaction between the user and the computer, controls hardware resources.
  • Popular OS: Windows, macOS, Linux.

2. Drivers

  • Software that controls specific hardware components (e.g., graphics card, printer).

3. Software Applications

  • Programs: Text editors, graphic editors, web browsers, games, etc.
  • Antivirus Software: To protect against malicious programs.
